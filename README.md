## Installing :bookmark_tabs:
- Crear carpeta en donde quieres que esté el proyecto
- Debes clonar el repositorio
```
https://github.com/thamaraRD/calculadora-app.git
```
- Instala todas las dependencias usando:
```
npm install
```
